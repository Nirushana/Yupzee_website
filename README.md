# Yupzee — Your university, finally connected.

Yupzee is a campus social network for students. Events, announcements, and your university community — all in one app that actually makes sense.

> **Status:** Launching soon. [Join the waitlist at yupzee.com](https://yupzee.com)

---

## What is Yupzee?

University life is scattered across emails, WhatsApp groups, Instagram accounts, and notice boards that no one checks. Students miss events, feel disconnected, and struggle to find their community — especially in first year.

Yupzee fixes that. One place for:

- **Campus Feed** — Real-time posts from friends, societies, and your university
- **Events** — Browse, RSVP, and get reminders. Never miss anything.
- **Follow & Connect** — Find your people on campus
- **Announcements** — Official updates that actually reach you

## This Repository

This is the **marketing website** for Yupzee, built with Next.js 16 (App Router), TypeScript, and Tailwind CSS v4.

### Pages

| Route | Description |
|---|---|
| `/` | Home — hero, features, waitlist CTA |
| `/features` | Feature deep-dives |
| `/universities` | Partner portal for universities |
| `/about` | Mission, values, story, team |
| `/blog` | Blog (placeholder) |
| `/legal/privacy` | Privacy Policy |
| `/legal/terms` | Terms & Conditions |
| `/legal/cookies` | Cookie Policy |

### Tech Stack

- [Next.js 16](https://nextjs.org) — App Router, static site generation
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS v4](https://tailwindcss.com)
- [Supabase](https://supabase.com) — waitlist form backend
